O comando document.getElementById('teste') é escrito em linguagem JavaScript. para eu mostrar o dogigo com a visão certa, basta coloca-lo ele entre a tag (CODE), Que chamamos de fonte mono espaçada (CODE) que é otima pra lermos código.
{
Caso você digite um código de alguma linguagem com identação (Segue em baixo)
A tag (CODE) não organiza seu Código com Identação, que é a organização do Código. (Então no site ficará como se fosse um texto, tudo escrito um do lado do outro.
Para que seja mostrado no site como Código voce envelopa tudo (Dês do CODE até o /CODE) e usa a Tag (pre) Que vai fazer com que seja considerado tudo, Os espaços na frente, as quebras de linha, as tabulações 
Caso você queira tirar as tubulações um jeito mais facil você segura (Shift+TAB)
Caso você queira acrecentar mais tubulações voce vai segura (TAB)
Tudo que estiver dentro da Tag (Pre) será considerado, seja com espaço ou quebras de linha.
}
